# Expedition YEG Website

A modern, responsive website built for a local bike tour company in Edmonton, Alberta. 
The brand focuses on being **authentic, locally owned, and operated by Edmontonians.**

## Features
- **Next.js App Router:** Fast and optimized React framework.
- **Tailwind CSS:** Modern utility-first styling.
- **Responsive Design:** Works on mobile, tablet, and desktop.
- **Tours & Booking:** Includes a catalog of local tours and a functional (mock) booking form.

## Backend & Management
- **Google Sheet (Inventory & Bookings):** Configured Google Sheet acting as the database for managing tour inventory and bookings.
- **Google Apps Script:** The backend logic is contained in `apps-script-backend.js` and should be deployed as a Web App.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
